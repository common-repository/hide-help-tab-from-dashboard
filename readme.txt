== Description ==
this plugin will hide help tab from wordpress dashboard.


== Installation ==
This section describes how to install the plugin and get it working.
e.g.
1. Upload `hide help tab for ever.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==
= Is there a way to re-display the help-tab after installing the plugin? =
yes, simply deactivate the plugin. you will see that the help-tab will appeare on your dashboard again.

= I installed the plugin. where is the plugin now? =
after you install the plugin, the help tab will be hidden automatically. there is no plugin management screen. if you want to re-display the help-tab, simply deactivate the plugin. you will see that the help tab will appear on your dashboard again.
